# Nhóm 1 - Xử Lý Ảnh Và Thị Giác Máy Tính - Anti Fake News Project

Dự án này là một hệ thống Trí tuệ Nhân tạo giúp nhận diện thông tin sai lệch (Fake News) từ văn bản, hình ảnh, hoặc các liên kết bài báo (URL) được phát triển cho nhóm 6 người.

Dự án đã được chia nhỏ thành cấu trúc mô-đun (Feature-based Architecture).

## 🚀 Hướng Dẫn Nhanh (Quick Start)
1. Cài đặt thư viện: `pip install -r requirements.txt`
2. Khởi chạy hệ thống: `python run_app.py`

## 📚 Tài Liệu Dự Án (Documentation)
Vui lòng xem chi tiết ở thư mục `docs/`:
- Kế hoạch & Kỹ thuật: Xem thư mục `docs/ke-hoach/`
- Phân công & Lịch trình (Cho 6 người): Xem thư mục `docs/phan-cong/`
