# 3. Lịch Trình và Quản Lý (Schedule & Management)

## 3.1. Giai đoạn 1: Base Architecture (Đã hoàn thành)
- Hoàn thiện cây thư mục `app/`, `src/`, `docs/`.
- File chạy tổng `run_app.py`.
- Tích hợp thành công 6 module chạy độc lập nhưng có khả năng nói chuyện với nhau.

## 3.2. Giai đoạn 2: Tối ưu (Đang tiến hành)
- Các thành viên tự mở rộng file của mình (ví dụ thêm hỗ trợ OCR vào `image_processor.py`, thêm API tìm kiếm Google Search vào `search_engine.py`).
- Merge code hàng ngày qua Git.

## 3.3. Giai đoạn 3: Báo cáo & Nghiệm thu (Tương lai)
- Bổ sung data mẫu vào thư mục `data/demo_cases/images`.
- Viết Unit Tests vào thư mục `tests/`.
- Thực hiện nghiệm thu dựa trên tài liệu `docs/ke-hoach/04-kiem-thu-va-mo-rong.md`.
